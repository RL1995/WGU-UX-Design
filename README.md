# User Experience Design - D479

This is a prototype website created for User Experience Design at Western Governors University (WGU).
- https://rl1995.github.io/WGU-UX-Design/

## Project Structure

- `pages/`
	- `index.html`
	- `about.html`
	- `dining.html`
	- `entertainment.html`
	- `lodging.html`
	- `transportation.html`
- `images/`
	- Page and social media image assets used by the HTML files
- `AllWireframes/`
	- Exported wireframe images for each page and site flow


## Tech Used

- HTML5
- Tailwind CSS (CDN)
- JavaScript

## Design Goals

- Mobile responsiveness
- Static header with persistent navigation
- Clear content hierarchy across all pages
